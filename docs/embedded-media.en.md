## Where Shopware’s media manager reaches its limits ...

... is where the `Embedded Media` entity comes into play. It is a combination of media, IFrame content, and videos, consisting of a cover, additional meta information, and a detailed description.

## How does it work?

### Overview

Via the main navigation in the admin: `Settings` → `Extensions` → `Embedded Media`

![](images/embedded-media-1.png)

![](images/embedded-media-2.png)

There are the following types of embedded media:

1. **Automatic assignment**
2. **Embedded** – e.g. IFrames
3. **Media** – e.g. videos from Shopware’s media manager
4. **Vimeo** – Vimeo videos with an embed link
5. **YouTube** – YouTube videos with an embed link
6. **Directory for 360 image viewers** – under construction

![](images/embedded-media-3.png)

### Additional settings

Depending on the type, additional settings can be configured, e.g. a cookie confirmation field for YouTube videos or external sources (IFrames).
